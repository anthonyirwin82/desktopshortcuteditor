I have provided builds for common systems for gtk2, qt5 and qt6 with the following exceptions:

Flatpak: When I looked into providing Flatpak packages I found that they run in a sandboxed environment so Flatpak is not well suited to a utility app like this that edits system files.

qt6 on aarch64: I have not provided a qt6 aarch64 build as the fpcupdeluxe cross compiler setup I use does not currently have the libqt6pas libraries for linking into the compiled binary. I will do a build for qt6 aarch64 when this is updated with the build pipeline tools I use.

Download DEB Packages

DEB builds tested on Ubuntu and Debian.

  • All versions work on Ubuntu 25.04 and newer providing you have the required libraries installed.
  • Ubuntu 24.04 LTS has the GTK2 and QT5 versions working because 24.04 LTS does not provide a package for libqt6pas6.
  • Debian 12 only the GTK2 version works as the libqt5pas1 library they have is not working with the latest qt5 builds.

amd64

  • desktop-shortcut-editor-1.0.0-amd64-qt5.deb
  • desktop-shortcut-editor-1.0.0-amd64-qt6.deb
  • desktop-shortcut-editor-1.0.0-amd64-gtk2.deb

arm64

  • desktop-shortcut-editor-1.0.0-arm64-qt5.deb
  • desktop-shortcut-editor-1.0.0-arm64-gtk2.deb

Download RPM Packages

RPM builds tested on Fedora 42, unfortunately the rpm build system didn't allow me to cross build the aarch64 RPMS for arm64 systems so only x86_64 builds are provided by me at this time. There is a generic tarball for aarch64 with builds for gtk2 and qt5.

  • desktop-shortcut-editor-1.0.0-1.qt5.x86_64.rpm
  • desktop-shortcut-editor-1.0.0-1.qt6.x86_64.rpm
  • desktop-shortcut-editor-1.0.0-1.gtk2.x86_64.rpm

Download Generic Tarballs

I have combined all the builds and an icon file into the tarball download for x86_64 and aarch64 architectures so you can choose which version you want to run.

  • desktop-shortcut-editor-1.0.0-x86_64-all.tar.gz
  • desktop-shortcut-editor-1.0.0-aarch64-all.tar.gz
